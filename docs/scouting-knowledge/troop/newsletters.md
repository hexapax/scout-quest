## Scoutbook Migration Instructions

Detailed step-by-step instructions were distributed for migrating from Teamsnap/Troopmaster to Scoutbook:

1. **Create accounts** on My.Scouting.Org (parent + Scout accounts)
2. **Enter contact information** in Scoutbook Plus profile (email, phone, opt-in for text alerts)
3. **Confirm advancement data** (merit badges, rank requirements, positions of responsibility)
4. **Log activities** (camping nights, hiking, service hours) via Activity Logs
5. **Learn messaging** via Scoutbook Legacy (email and text to troop contacts)

Key notes:
- Camping night logging required for Scouts who haven't earned Camping MB
- Service hours that aren't pre-listed need to be added by Nicole Allen first
- Text messages come from a do-not-reply service (like weather alerts, not group chat)
